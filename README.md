# SmartExpenseTracker

## Description
A simple Java console application that helps users manage and divide their monthly salary into multiple categories such as bills, food, and savings.  
The program takes the salary and expenses as input and then calculates the total allocated amount and remaining balance.  

---

## Features
- Input monthly salary from the user  
- Divide money into categories:
  - Bills  
  - Food  
  - Transportation  
  - Entertainment  
  - Shopping  
  - Savings  
- Calculate total expenses and remaining balance  
- Save every run summary into a note file (`records.txt`)  
- Keep a cumulative total of all **savings** and **remaining balance** in a separate file (`savings.txt`)
