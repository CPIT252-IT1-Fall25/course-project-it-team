#  Smart Expense Tracker  


## Project Overview
**Smart Expense Tracker** is a JavaFX desktop application designed to help users manage their daily expenses.  
The app allows users to log in, add and categorize expenses, and view summaries (daily, weekly, monthly).  
This project demonstrates the use of **Object-Oriented Programming (OOP)** and several **Design Patterns** learned in the course.
